# Roadmap

Themes only — a high-level view of where Aegis is heading. Detailed release
sequencing and coverage gates are tracked in the maintainers' planning workspace.

## At a glance (as of v0.0.8)

- **Shipped:** v0.0.1 → v0.0.8.
- **Active hosts (3):** Claude Code, OpenCode, Codex.
- **v0.0.8:** full template coverage (20 HTML + 14 Markdown + 16 JSON) and
  multi-format output selection (per-kind format index + format-aware
  `${TEMPLATE}` resolver + widened user choice).
- **Next — v0.1.0:** stable plugin contract + 1.0 user-facing docs across the 3 active hosts.
- **Deferred (~v0.5.0):** Cursor + Zed projections, grouped with Trae,
  Antigravity, and GitHub Copilot.
