# Roadmap

Themes only — a high-level view of where Aegis is heading. Detailed release
sequencing and coverage gates are tracked in the maintainers' planning workspace.

## At a glance

- **Active hosts (3):** Claude Code, OpenCode, Codex.
- **Public baseline (v0.1.x):** stable plugin contract, full template coverage
  (20 HTML + 14 Markdown + 16 JSON), and multi-format output selection
  (per-kind format index + format-aware `${TEMPLATE}` resolver + widened user
  choice) across the 3 active hosts.
- **Deferred (~v0.5.0):** Cursor + Zed projections, grouped with Trae,
  Antigravity, and GitHub Copilot.
