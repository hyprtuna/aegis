---
name: aegis-skill-pin
description: 'Pin a skill so it surfaces in the slash menu Pinned section'
---

Pin a named skill to your per-user shortcuts list.

Pinned skills live in `~/.aegis/pins.json` and appear in the "Pinned" section at the top of the skill list. The default cap is 5 — unpin a skill first if you need to add another past the cap.
